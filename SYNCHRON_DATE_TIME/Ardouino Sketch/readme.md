                                            Circuit Board Components

- 9 Pin cable(Male -Male).
- 5 Pin cable(Male -Female).

---------------------------------------------------------------

- (SCK) -> D13
- (MOSI) -> D11
- (MISO) -> D12
- (VCC) ->  5V
- (GND) -> GND
- (LED1) -> D3
- (LED2) -> D8
- (Button1) -> D7
- (Button2) -> D9
