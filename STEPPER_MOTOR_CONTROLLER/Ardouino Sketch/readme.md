                                            Circuit Board Components 
-----------------------------------------------------------

- Stepper Motor 28BYJ-48
- ULN2003A
- 6 Pin cable.
- Ardouino Uno. 
- USB Connection cable 0.4m (type B to type A)
- 1 Pushbutton



---------------------------------------------------------------

- (-) -> GND
- (+) -> 5V

---------------------------------------------------------------

- (IN1) -> D8
- (IN2) -> D9
- (IN3) -> D10
- (IN4) -> D11
